# Decision Loop

1. Frame the real decision, deadline, values, constraints, and no-action option.
2. Separate facts, estimates, preferences, and unknowns.
3. Generate at least two real alternatives; include a reversible experiment when possible.
4. Compare benefits, costs, risks, affected people, reversibility, and regret.
5. Ask what evidence would change the ranking.
6. Return a decision brief. The user chooses.

Stop when the user has a sufficiently clear choice, decides to defer, or needs qualified professional advice. Do not optimize a proxy score into a life verdict.
