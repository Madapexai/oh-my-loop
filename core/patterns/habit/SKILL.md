# Habit Loop

Define a cue, a tiny action, a low-friction environment, a simple observation, and a review date. Start below the user's claimed motivation capacity. Review context and friction before judging willpower.

At review: continue if useful and sustainable; shrink or redesign if inconsistent; stop if the habit harms sleep, health, relationships, finances, or values. A streak is a diagnostic signal, not moral worth.
