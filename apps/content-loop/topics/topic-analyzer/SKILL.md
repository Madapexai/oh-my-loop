# Topic Analyzer — 选题深度分析

将一个模糊的选题方向变成精确的创作蓝图。

## 分析框架（5维度）

### 1. 平台适配分析
基于选题特性，推荐最适合的平台：

| 选题特征 | 推荐平台 | 理由 |
|---------|---------|------|
| 深度技术/方法论 | 公众号 + 知乎 + 掘金 | 长文适合深度展开 |
| 实用工具/效率技巧 | 小红书 + 公众号 | 视觉化展示效果好 |
| 行业观点/趋势判断 | 即刻 + X/Twitter + LinkedIn | 短观点引发讨论 |
| 个人故事/经历 | 公众号 + 小红书 | 情感共鸣强 |
| 数据/研究报告 | 公众号 + 知乎 + LinkedIn | 专业性展示 |

### 2. 受众画像
- **核心受众**：谁最关心这个话题？（年龄、职业、痛点）
- **次级受众**：谁可能被标题吸引进来？
- **传播受众**：谁会转发/分享这类内容？

### 3. 情绪切入点（九大共鸣元素）
选择1-2个主要情绪切入点：
1. 💰 **金钱** — 与收入、财富、省钱相关
2. ❤️ **关系** — 两性、亲子、职场人际关系
3. ⚡ **冲突** — 争议、对立观点
4. 💀 **健康/安全** — 生命安全、健康焦虑
5. 🚀 **捷径** — 快速达成目标的方法
6. 🤯 **反常识** — 打破认知的信息
7. ❓ **悬念** — 好奇心驱动
8. 🏠 **家国/归属** — 集体认同感
9. 😰 **焦虑/恐惧** — 不安全感

**敏感元素使用边界：**
- 💰 金钱：可用，但避免过度渲染收入（平台可能判定"炫富"）
- ❤️ 关系：可用，但避免露骨内容
- ⚡ 冲突：可用，但避免人身攻击和站队引战
- 💀 健康/安全：可用，但避免制造恐慌（医疗内容需标注"仅供参考"）
- 😰 焦虑/恐惧：可用，但要提供解决方案，不能纯贩卖焦虑
- 避免直接涉及：政治、色情、暴力、谣言

### 4. 竞品分析（自动执行）

自动用WebSearch搜索同类话题：
1. 搜索 "公众号 [关键词]" 找微信文章
2. 搜索 "知乎 [关键词]" 找知乎回答
3. 搜索 "小红书 [关键词]" 找小红书笔记
4. 分析前5篇竞品：标题公式、结构、数据引用、评论区反馈
5. 找到竞品的**不足之处**作为自己的差异化切入点

### 5. 内容结构推荐
基于选题类型推荐最佳结构：

| 选题类型 | 推荐结构 | 适用场景 |
|---------|---------|---------|
| 问题解决型 | SCQA模型 | 教程、方法论、工具介绍 |
| 经验分享型 | 故事+金句+CTA | 个人经历、复盘总结 |
| 知识科普型 | 清单体/编号型 | 技术科普、行业知识 |
| 观点表达型 | 争议讨论体 | 行业观点、趋势判断 |
| 产品测评型 | Before/After对比 | 工具测评、方法对比 |

## 输出格式

分析完成后输出：

```markdown
## 选题分析报告：[选题]

### 平台推荐
首选：[平台] | 次选：[平台]

### 受众画像
核心受众：[描述]
传播受众：[描述]

### 情绪切入点
主切入点：[元素] — [具体角度]
辅助切入点：[元素] — [具体角度]

### 差异化角度
竞品写了什么：[概括]
我们的不同：[差异化点]

### 推荐标题（5选1）
1. [数字量化型]
2. [悬念提问型]
3. [矛盾冲突型]
4. [反差对比型]
5. [痛点直击型]

### 推荐结构
[结构名称] — [理由]

### 创作要点
- 必须包含：[关键素材/数据]
- 避免踩坑：[常见错误]
- 预期字数：[建议范围]
```
