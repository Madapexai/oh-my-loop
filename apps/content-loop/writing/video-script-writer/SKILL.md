# Video Script Writer — 视频脚本撰写与自动化生产

从选题到完整视频脚本，覆盖短视频、中视频、长视频全链路。集成业界最佳实践，支持自动化视频生产流水线。

## 一、视频类型与脚本结构

### 类型A：短视频（15-60秒）— 抖音/快手/视频号/Reels/Shorts

**黄金3秒法则**：前3秒必须出现钩子，否则用户划走

**标准结构（15秒版）**：
```
0-3秒  钩子（视觉冲击/反常识/痛点提问）
3-8秒  核心信息（一个观点/一个技巧/一个反转）
8-12秒 证明/演示（数据/案例/前后对比）
12-15秒 CTA（关注/点赞/评论引导）
```

**标准结构（60秒版）**：
```
0-3秒   钩子（必须抓住注意力）
3-15秒  问题/痛点展开（让用户产生共鸣）
15-35秒 解决方案/核心内容（干货输出）
35-50秒 案例/效果展示（证明有效性）
50-60秒 CTA + 金句结尾（引导互动）
```

**钩子公式（10种）**：
1. **反常识型**："你以为的XX，其实完全错了"
2. **数字冲击型**："我分析了1000个爆款，发现1个规律"
3. **痛点直击型**："为什么你天天XX还是XX？"
4. **悬念型**："最后一种方法，90%的人都不知道"
5. **对比型**："Before vs After，3天变化"
6. **挑战型**："挑战XX天做XX，第X天"
7. **揭秘型**："行业内幕：XX不会告诉你的事"
8. **情绪型**："救命！这也太绝了吧！"
9. **身份标签型**："适合学生党的XX"
10. **场景代入型**："凌晨2点，我还在..."

### 类型B：中视频（3-10分钟）— B站/YouTube/知识类

**结构模板**：
```
0-30秒   开场钩子（问题/冲突/反常识）
30-60秒  背景介绍（为什么这个话题重要）
1-3分钟  核心内容展开（分2-3个点）
3-5分钟  案例/演示/实操（具体展示）
5-7分钟  进阶技巧/常见误区（深度内容）
7-9分钟  总结回顾（核心要点提炼）
9-10分钟 CTA（关注+下期预告）
```

### 类型C：长视频（10-30分钟）— 纪录片/深度解析/课程

**结构模板**：
```
0-1分钟   片头钩子（核心问题/震撼画面）
1-3分钟   背景铺垫（历史/现状/数据）
3-8分钟   第一部分：问题分析
8-15分钟  第二部分：深度展开
15-22分钟 第三部分：案例/证据
22-27分钟 第四部分：解决方案/观点
27-29分钟 总结升华
29-30分钟 片尾CTA
```

## 二、分镜设计规范

### 分镜表格式

| 镜号 | 时间 | 景别 | 画面内容 | 台词/字幕 | 音效/BGM | 备注 |
|------|------|------|----------|-----------|----------|------|
| 1 | 0-3s | 特写 | 产品特写+文字弹出 | "90%的人都不知道" | 紧张音效 | 钩子 |
| 2 | 3-8s | 中景 | 主持人出镜 | "今天教你一个方法" | BGM渐入 | 过渡 |

### 景别定义
- **特写**：聚焦细节，制造冲击力
- **近景**：人物胸部以上，适合口播
- **中景**：人物膝盖以上，适合动作展示
- **全景**：完整场景，适合环境交代
- **远景**：大环境，适合开场/结尾

### 转场方式
- **硬切**：最常用，节奏快
- **淡入淡出**：柔和过渡
- **划像**：方向感强
- **缩放转场**：聚焦感
- **遮罩转场**：创意感

## 三、口播稿写作规则

### 语言风格
- **口语化**：像面对面聊天，不用书面语
- **短句为主**：每句不超过15个字，方便断句呼吸
- **节奏感**：每3-5句一个停顿，给用户消化时间
- **互动感**：多用"你"、"你知道吗"、"想象一下"

### 禁止清单
- ❌ "综上所述"、"由此可见"、"值得注意的是"
- ❌ 长复合句（超过20字）
- ❌ 抽象概念无具体例子
- ❌ 平铺直叙无情绪波动

### 标注规范
```
【画面】描述画面内容
【字幕】显示在屏幕上的文字
【音效】音效提示
【转场】转场方式
【BGM】背景音乐提示
【停顿】停顿1秒
```

## 四、平台算法适配

### 抖音/快手
- 完播率 > 点赞率 > 评论率 > 转发率
- 前3秒完播率决定推荐量
- 视频时长：15-30秒最佳（新手），60秒内（进阶）
- 发布频率：每天1-3条

### B站
- 完播率 + 三连率（点赞+投币+收藏）
- 弹幕互动是核心指标
- 视频时长：5-15分钟最佳
- 发布频率：每周1-3条

### YouTube
- 观看时长（Watch Time）最重要
- CTR（点击率）和 AVD（平均观看时长）
- 视频时长：8-15分钟最佳
- 发布频率：每周1-2条

### 视频号
- 社交推荐 > 算法推荐
- 完播率和转发率最重要
- 视频时长：30-60秒最佳
- 发布频率：每天1-2条

## 五、从文章到视频的改编流程

```
输入：一篇已发布的文章
  ↓
Step 1: 提取核心观点 — 找出1-3个最有价值的观点
  ↓
Step 2: 设计钩子 — 将最反常识/最震撼的观点做成开头
  ↓
Step 3: 拆解结构 — 按视频时长重新组织内容
  ↓
Step 4: 口语化改写 — 将书面语改为口播语
  ↓
Step 5: 添加画面标注 — 标注每个时间点的画面需求
  ↓
Step 6: 添加字幕和音效 — 标注字幕内容和音效提示
  ↓
Step 7: 输出完整脚本 — 含时间轴、口播稿、画面、字幕
```

## 六、自动化视频生产流水线

### 6.1 整体架构

```
┌─────────────────────────────────────────────────────────────────┐
│                    自动化视频运营流水线                           │
├─────────────────────────────────────────────────────────────────┤
│  选题层 → 脚本层 → 素材层 → 生成层 → 剪辑层 → 发布层 → 数据层    │
└─────────────────────────────────────────────────────────────────┘
```

### 6.2 各层详解

#### 选题层 — 热点发现与选题策划
| 工具/方法 | 说明 | 适用场景 |
|-----------|------|----------|
| 抖音热点宝 | 查看实时热点、低粉爆款 | 短视频选题 |
| 微博热搜 | 社会热点、娱乐话题 | 时效性内容 |
| 知乎热榜 | 深度话题、知识类 | 中视频选题 |
| Google Trends | 全球趋势、长尾词 | YouTube选题 |
| AI选题助手 | 基于历史数据推荐 | 全平台 |

**选题公式**：
- 科普类："XX是什么" + "为什么重要" + "怎么用"
- 软件介绍："痛点" + "解决方案" + "实操演示"
- 金融类："数据/新闻" + "解读" + "影响分析" + "建议"

#### 脚本层 — 脚本生成与分镜设计
- 使用本 skill 生成完整口播稿
- 自动拆分分镜，标注时间轴
- 输出画面需求清单（素材类型、时长、风格）

#### 素材层 — 素材获取与管理
| 素材类型 | 来源 | 工具 |
|----------|------|------|
| 视频素材 | Pexels / Pixabay / Coverr | MoneyPrinterTurbo 内置 |
| AI生成素材 | FLUX-1 / Midjourney / 可灵 / Runway | API调用 |
| 屏幕录制 | OBS / 系统录屏 | 软件演示类 |
| 数据可视化 | 图表库 / HyperFrames | 金融/科普类 |
| 音乐/音效 | Epidemic Sound / 爱给网 | BGM库 |

#### 生成层 — AI视频生成
| 工具 | 类型 | 特点 | 适用场景 |
|------|------|------|----------|
| **MoneyPrinterTurbo** | 开源自动化 | 一键生成，素材拼接 | 批量短视频 |
| **HyperFrames** | HTML→MP4 | 确定性渲染，适合Agent | 数据可视化/产品视频 |
| **可灵 (Kling)** | 文生视频 | 国产，运镜控制强 | 创意片段 |
| **Runway Gen-4** | 文生/图生视频 | 行业标杆 | 高质量创意视频 |
| **Pika 2.0** | 文生视频 | 特效丰富 | 风格化视频 |
| **HeyGen** | 数字人 | 口播视频 | 讲解类/新闻类 |
| **Sora 2** | 文生视频 | OpenAI，物理真实 | 电影级片段 |

#### 剪辑层 — 视频合成与后期
| 工具 | 类型 | 功能 |
|------|------|------|
| **FFmpeg** | 命令行 | 视频合并、转码、滤镜、字幕 |
| **剪映/CapCut** | 桌面/移动端 | 智能剪辑、自动字幕、模板 |
| **HyperFrames CLI** | 命令行 | HTML渲染为MP4 |
| **Python + MoviePy** | 编程 | 自动化剪辑流水线 |

**FFmpeg 常用命令**：
```bash
# 合并视频片段
ffmpeg -f concat -i filelist.txt -c copy output.mp4

# 添加字幕
ffmpeg -i input.mp4 -vf "subtitles=subtitle.srt" output.mp4

# 添加背景音乐
ffmpeg -i video.mp4 -i music.mp3 -c:v copy -c:a aac -shortest output.mp4

# 调整分辨率
ffmpeg -i input.mp4 -vf "scale=1080:1920" output.mp4

# 视频加速/减速
ffmpeg -i input.mp4 -vf "setpts=0.5*PTS" -af "atempo=2.0" output.mp4
```

#### 发布层 — 多平台自动发布
| 平台 | 工具 | 说明 |
|------|------|------|
| 抖音 | 抖音开放平台API | 需申请权限 |
| B站 | bilibili-api | Python库 |
| YouTube | YouTube Data API | Google官方 |
| 视频号 | 微信开放平台 | 需企业认证 |
| 小红书 | 手动/第三方工具 | 限制较多 |

#### 数据层 — 数据分析与优化
- 播放量、完播率、互动率追踪
- A/B测试不同标题/封面
- 评论区 sentiment 分析
- 迭代优化脚本模板

## 七、主题-specific 视频方案

### 7.1 科普类视频

**选题方向**：
- "XX原理3分钟讲清楚"
- "每天一个冷知识"
- "科学家不会告诉你的XX"

**脚本模板**：
```
0-3秒   钩子：反常识问题（"你知道为什么天空是蓝色的吗？"）
3-15秒  背景：现象描述，引发好奇
15-40秒 原理：分步骤讲解，配合动画/图示
40-55秒 应用：生活中的实际例子
55-60秒 CTA："关注我，每天学一个科学知识"
```

**推荐工具链**：
- 素材：HyperFrames（数据可视化）+ Pexels
- 生成：MoneyPrinterTurbo 批量生成
- 配音：Kokoro TTS / ElevenLabs

### 7.2 软件介绍/教程类视频

**选题方向**：
- "XX软件5个隐藏功能"
- "新手必看的XX教程"
- "XX vs YY 对比测评"

**脚本模板**：
```
0-3秒   钩子：痛点（"还在手动做XX？太麻烦了"）
3-10秒  产品介绍：是什么，解决什么问题
10-40秒 功能演示：录屏+标注，核心功能
40-55秒 效果展示：Before vs After
55-60秒 CTA："评论区领取使用手册"
```

**推荐工具链**：
- 录屏：OBS / 系统录屏
- 标注：剪映自动字幕 + 箭头标注
- 剪辑：FFmpeg + Python脚本

### 7.3 金融类视频

**选题方向**：
- "今日股市3分钟复盘"
- "XX政策解读"
- "小白理财入门"

**脚本模板**：
```
0-3秒   钩子：数据冲击（"今天A股跌了3%，但这类人却赚了"）
3-15秒  市场概况：关键指数/数据
15-40秒 深度分析：原因解读，图表展示
40-55秒 观点/建议：风险提示
55-60秒 CTA："不构成投资建议，仅供参考"
```

**推荐工具链**：
- 数据：AKShare / Tushare（A股数据API）
- 图表：HyperFrames / ECharts
- 生成：MoneyPrinterTurbo + 自定义模板
