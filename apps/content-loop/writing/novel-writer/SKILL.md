# Novel Writer — 小说创作

从世界观设定到连载发布，覆盖长篇小说创作全链路。

## 创作流程（六步流水线）

参考 AI_NovelGenerator 和 91Writing 开源项目：

```
Step 1: 世界观设定 → Step 2: 分卷大纲 → Step 3: 章节目录
  → Step 4: 正文草稿 → Step 5: 一致性审校 → Step 6: 迭代改写
```

### Step 1: 世界观设定

**核心要素**：
- **时代背景**：古代/现代/未来/架空
- **世界观规则**：魔法体系/科技水平/社会结构
- **核心冲突**：主角面临的最大矛盾
- **主题**：想表达的核心思想

**世界观文档模板**：
```markdown
## 世界观设定：[作品名]

### 基础设定
- 时代：[具体年代/架空设定]
- 地点：[主要场景]
- 规则：[世界运行的核心规则]

### 势力分布
- [势力A]：[描述]
- [势力B]：[描述]

### 核心冲突
- 表面冲突：[描述]
- 深层冲突：[描述]

### 主题
- 核心主题：[一句话]
- 次要主题：[一句话]
```

### Step 2: 分卷大纲

**三幕式结构**：
```
第一幕： setup（设定）— 约占20%
  - 介绍主角、世界、日常
  - 触发事件：打破日常的事件
  - 第一转折点：主角决定行动

第二幕： confrontation（对抗）— 约占60%
  - 上升行动：主角行动，遇到阻碍
  - 中点：局势发生重大变化
  - 下降行动：最大危机出现
  - 第二转折点：最低谷，似乎要失败

第三幕： resolution（解决）— 约占20%
  - 高潮：最终对决
  - 结局：问题解决，新世界建立
```

### Step 3: 章节目录

**章节规划表**：
| 卷 | 章节 | 核心事件 | 爽点/钩子 | 字数目标 |
|----|------|---------|----------|---------|
| 第一卷 | 第1-10章 | 触发事件 | 第3章小高潮 | 2万 |
| ... | ... | ... | ... | ... |

### Step 4: 正文草稿

**黄金三章法则**（决定作品生死）：

**第一章核心任务**：50字内建立冲突，300字内建立代入感

**各类型开头公式**：

| 类型 | 开头公式 | 案例 |
|------|---------|------|
| 言情/甜宠 | 前世惨死→重生回关键节点→发誓改命 | 《重生之将门毒后》 |
| 灵异/悬疑 | 看似正常→某个细节不对劲→恐惧递进 | — |
| 逆袭/励志 | 低谷开局→获得金手指→第一次反击 | 《二号首长》 |
| 玄幻/仙侠 | 世界观残酷→废柴身份→获得奇遇 | 《完美世界》 |
| 都市/职场 | 平凡日常→意外事件→命运转折 | — |

**人物设定模板**：

**主角**：
- 表面特征：[外在形象]
- 隐藏特质：[不为人知的秘密/能力]
- 核心欲望：[最想要什么]
- 最大恐惧：[最怕失去什么]
- 成长弧线：[从什么状态到什么状态]

**反派**：
- 动机：[为什么做坏事]
- 能力：[给主角制造什么麻烦]
- 弱点：[最终如何被击败]

### Step 5: 一致性审校

**审校清单**：
- [ ] 角色设定一致（名字、外貌、能力不矛盾）
- [ ] 时间线一致（事件顺序合理）
- [ ] 世界观一致（规则不前后矛盾）
- [ ] 伏笔回收（埋下的伏笔有交代）
- [ ] 语气一致（叙述风格统一）

**角色状态追踪表**：
```markdown
| 角色 | 当前状态 | 能力等级 | 关系变化 | 最新出场 |
|------|---------|---------|---------|---------|
| 主角 | [状态] | [等级] | [关系] | 第X章 |
| 反派A | [状态] | [等级] | [关系] | 第X章 |
```

### Step 6: 迭代改写

**改写重点**：
- 节奏：删减拖沓段落，加快关键情节
- 爽点：增加打脸/反转/高潮的密度
- 代入感：增强感官描写和内心独白
- 钩子：每章结尾留悬念

## 节奏控制

| 类型 | 爽点密度 | 反转频率 | 单章字数 |
|------|---------|---------|---------|
| 甜宠文 | 每3-5章一个甜蜜互动 | 每5章一个中型反转 | 2000-2500 |
| 爽文 | 每2-3章一个小打脸 | 每10章一个大打脸 | 2000-3000 |
| 悬疑文 | 每3章展现一次特殊能力 | 每5章推进一个关键线索 | 2000-2500 |
| 玄幻文 | 每3-5章一场战斗 | 每20章一场大战 | 2500-3500 |

## 平台规则与发布策略

### 番茄小说
- **签约线**：2万字申请签约
- **推荐机制**：3万字推书荒 → 8万字推荐评估 → 10万字正式推荐
- **更新频率**：每天1-3章，每章2050-2300字
- **全勤奖**：每月稳定10-12万字
- **算法核心**：完读率 → 追更率 → 互动率 → 留存率

### 起点中文网
- **投稿**：6000字直发或内投编辑邮箱
- **男频**：2000-3000字/章
- **女频**：1500-2500字/章
- **更新时间**：工作日上午（编辑在线率高）

### 晋江文学城
- **题材**：言情、耽美、百合为主
- **字数**：30万字以上容易入V
- **更新**：日更3000+有榜单优势

### Wattpad（海外）
- **节奏**：慢热，注重情感铺垫
- **互动**：评论区互动很重要
- **付费**：Wattpad Stars计划

### Royal Road（海外Web小说）
- **类型**：玄幻、游戏、系统流为主
- **更新**：周更3-5章
- **社区**：读者评论直接影响推荐

### Amazon KDP
- **格式**：电子书+纸质书
- **定价**：$2.99-$9.99（70%版税）
- **推广**：KDP Select（90天独家，Kindle Unlimited分成）

## 书名与简介公式

### 书名公式
- **悬念型**：《我在XX的那些年》《XX不会告诉你》
- **反差型**：《从XX到XX》《重生之XX》
- **身份型**：《XX师》《XX传》《XX录》
- **情感型**：《一生一世XX》《XX年XX》
- **系统型**：《XX：从XX开始》《开局XX》

### 简介公式
```
[主角身份] + [核心冲突] + [金手指/独特设定] + [悬念钩子]

示例：
"沈妙是相府嫡女，却被庶妹陷害，三尺白绫下立毒誓。
重生回十四岁，她手握前世记忆，誓要改写命运。
只是那个冷面王爷，为什么总是出现在她身边？"
```
