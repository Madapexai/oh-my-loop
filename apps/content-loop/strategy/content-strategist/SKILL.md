# Content Strategist — 内容策略与矩阵规划

从0到1搭建内容体系，让每一步都有战略意图。

## 内容战略设计框架

### 1. 定位三角

```
        你擅长什么？
            △
           / \
          /   \
         /     \
        /   甜   \
       /    蜜    \
      /     区     \
     /_____________\
用户需要什么？    竞品没做什么？
```

**甜蜜区** = 你擅长的 × 用户需要的 × 竞品没做的

### 3. 内容分层体系

**金字塔内容模型**：
```
                    ┌─────────────┐
                    │  旗舰内容     │  10%
                    │  深度长文/课程 │  建立权威
                    ├─────────────┤
                    │  支柱内容     │  30%
                    │  系列文章/视频 │  建立体系
                    ├─────────────┤
                    │  日常内容     │  60%
                    │  短内容/互动  │  维持活跃
                    └─────────────┘
```

**内容类型配比**：
| 类型 | 比例 | 目的 | 示例 |
|------|------|------|------|
| 教育型 | 40% | 建立专业信任 | 教程、方法、知识 |
| 娱乐型 | 30% | 增加传播性 | 故事、趣闻、段子 |
| 互动型 | 20% | 增强参与感 | 问答、投票、讨论 |
| 转化型 | 10% | 直接变现 | 产品、优惠、活动 |

### 4. 竞品差异化策略

**差异化定位方法**：
1. **人群差异化**：同样话题，面向不同人群
   - 示例：别人做"职场技能"，你做"35+职场转型"

2. **形式差异化**：同样内容，用不同形式呈现
   - 示例：别人写图文，你做视频+图文

3. **角度差异化**：同样话题，从不同角度切入
   - 示例：别人讲"怎么成功"，你讲"怎么优雅地失败"

4. **深度差异化**：同样话题，做到更深更细
   - 示例：别人讲"怎么写文案"，你讲"怎么写医疗器械文案"

### 5. 长期规划框架

**12个月内容路线图**：

```
Month 1-2: 定位期
  - 确定定位、平台、内容方向
  - 发布20-30篇内容测试
  - 收集数据反馈

Month 3-4: 建立期
  - 找到表现最好的内容类型
  - 建立内容模板和SOP
  - 开始系列化内容

Month 5-6: 增长期
  - 加大发布频率
  - 尝试跨平台分发
  - 开始初步变现测试

Month 7-8: 优化期
  - 数据分析，优化低效环节
  - A/B测试标题、封面、时间
  - 建立内容资产库

Month 9-10: 变现期
  - 推出第一款付费产品
  - 建立私域流量池
  - 测试多种变现方式

Month 11-12: 扩张期
  - 放大有效策略
  - 考虑团队化/工具化
  - 规划下一年方向
```
